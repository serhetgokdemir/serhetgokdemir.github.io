Building Neural Networks and Custom Convolutions: Deep Learning Techniques

This project explores the foundational techniques of deep learning, including neural network construction and custom convolution operations.

It begins with implementing a classification task using PyTorch and PyTorch Lightning, highlighting the development of custom architectures, hyperparameter tuning, and training scripts.

In the second phase, 1D and 2D convolution functions have developed from scratch, applied to signal and image data. The 1D convolution function was used to denoise a sine wave. The 2D convolution function was applied to grayscale and RGB images by using the effects of blurring and Sobel operator-based edge detection.Click here to see the project on Github.
